## рҹ—„пёҸ Ma'lumotlar bazasi (вң… tasdiqlangan, yaratilgan)

**Format:** 2 ta liga (LaLiga, Premier Liga), har birida 20 ta o'yinchi, round-robin (har kim har kim bilan 2 marta), 38 kunlik (matchday) turnir. Natijalarni o'yinchining o'zi kiritadi, raqibi tasdiqlaydi/rad etadi.

| Jadval | Maydonlar | Izoh |
|---|---|---|
| `users` | id, telegram_id, nickname, language, registered_at | |
| `leagues` | id, name, max_players, status | name: "LaLiga"/"Premier Liga" |
| `registrations` | id, user_id, league_id, registered_at | UNIQUE(user_id) вҖ” bitta user faqat bitta ligaga |
| `matches` | id, league_id, matchday, player1_id, player2_id, score1, score2, submitted_by, status | matchday: 1-38, status: pending/awaiting_confirmation/confirmed/rejected |
| `prizes` | league_id, top_scorer_user_id, winner_user_id | Oltin Butsa (eng ko'p gol), Oltin To'p (g'olib) |
| `wc_registrations` | id, user_id (UNIQUE), group_letter, team_name, registered_at | World Cup ro'yxati вҖ” liga'dan ALOHIDA (foydalanuvchi ikkalasida qatnasha oladi). UNIQUE(user_id): WC'da 1 marta |
| `wc_groups` | group_letter (PK), draw_date, last_notified_matchday | World Cup guruh holati. draw_date: guruh 4 jamoaga to'lib o'yinlar yaratilgan vaqt (matchday-lock uchun) |
| `wc_matches` | id, group_letter, matchday, player1_id, player2_id, score1, score2, submitted_by, status | World Cup guruh o'yinlari (liga matches kabi). 4 jamoa вҶ’ 6 o'yin, 3 matchday |

**Jadval generatsiyasi:** Liga 20 kishiga to'lganda, `schedule.py` dagi circle method algoritmi avtomatik 380 ta match yaratadi (2 Г— 19 tur = 38 matchday).
